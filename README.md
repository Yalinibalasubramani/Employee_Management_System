The Employee Management System (EMS) is a simple Java-based console application designed to manage employee records.
It allows the user to add, view, update, and remove employee details efficiently.
All records are stored in individual text files, ensuring data persistence even after the program is closed.

Features:
Add new employee details
View existing employee information
Update employee details (name, contact, position, etc.)
Remove an employee record
Automatically saves each record as a text file (file<ID>.txt)
Simple menu-driven interface for ease of use

Technology Used:
Language: Java
IDE / Editor: Visual Studio Code
Storage: File handling (.txt files)

Concepts Used:
Object-Oriented Programming (OOP)
File I/O (Input/Output Streams)
Exception Handling
Scanner Class for user input
